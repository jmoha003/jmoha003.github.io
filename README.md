## How to Update Content

1. Go to https://github.com/jmoha003/jmoha003.github.io
2. Click `index.html`
3. Click the pencil icon (edit)
4. Make changes
5. Click **Commit changes**
6. Wait 1-2 minutes for GitHub Pages to rebuild

## Common Updates

### Update citation count
Search for `data-target="69"` and the fallback text `69` near it. Change both to the new number.

### Add a new publication
Find the publications section. Copy an existing `<div class="pubr">` block, paste it above the others, and update the title, venue, and year.

### Update resume
Upload the new PDF to the repo with the same filename `JoshitMohanty_Resume.pdf`. The download link stays the same.

### Add a new award
Find the awards section. Copy an existing `<div class="aw">` block, paste it, and update year and text.

### Add a new press mention
Find the press section. Copy an existing `<a href="..." class="pl">` block, paste it, and update the URL, logo domain, name, and year.

### Update sitemap date (optional, for major changes only)
Open `sitemap.xml`, change `<lastmod>` to today's date in YYYY-MM-DD format.

## Logo API

Logos load from Logo.dev using a free API token. Free tier allows 500,000 requests/month.

To add a new logo to the scrolling strip, add this inside the `<div class="lt">` block (in both the original and duplicate set):

```html
<img src="https://img.logo.dev/DOMAIN.com?token=pk_EiCQ8G9jTi61mdG9lUWoKQ&format=png" alt="NAME" title="FULL NAME">
```

Replace `DOMAIN.com` with the organization's website domain.

## If Something Breaks

- **Site not loading:** Check GitHub repo Settings → Pages. Source should be `main` branch, `/ (root)`.
- **Domain not resolving:** Check Cloudflare DNS records (4 A records + 1 CNAME).
- **Logos not loading:** Logo.dev may be down or token expired. Sign into https://logo.dev to check.
- **Resume not downloading:** Confirm `JoshitMohanty_Resume.pdf` exists in the repo root.
